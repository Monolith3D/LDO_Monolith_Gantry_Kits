## Known Issues

Batch 1 kits have excessive thread engagement length for the steppers. This can cause problems with steppers that have more limited hole depths than the most common 48mm body LDO steppers. Please use M3 spring washers (or any other small OD M3 washers) in the stepper mounting hole counterbores to avoid this issue.

We’re working with LDO to get these washers to your resellers. We're trying to include the washers in batch 1 kits that haven’t shipped yet.
