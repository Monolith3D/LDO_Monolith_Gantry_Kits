## What's this?
**These are the official Milled [Monolith Gantry](https://github.com/Monolith3D/Monolith_Gantry) kits for Voron V2 and Trident, manufactured by [LDO](https://ldomotion.com/).**

## Features

* Metal Monolith in an easy-to-assemble, aesthetically pleasing package
* 2WD and AWD options in one kit to provide a convenient upgrade path
* Everything included except belts and steppers
* Robust, removable tensioners for easier belting up and pre-tensioning
* 9/10mm belt support (toolhead dependent)
* Genuine Gates pulleys and press-fit live shaft idlers
* AWD front extrusion support for toolchangers

> [!NOTE]
> **The flanged bearing holes in Batch 1 were machined to an incorrect depth.**
> 
> **A mistake that was fixed with additional shims. Learn more about it [here](https://github.com/Monolith3D/LDO_Monolith_Gantry_Kits/tree/main/Images/Batch_1).**

## Recommendations

* Made for standard 250-350mm bed sizes
* Frame upgrades for full performance potential (4040, structural panels)
* Best paired with non-modular, performance-oriented toolheads, preferably metal
* LDO 2504 (S45R) Speedy Power steppers (45mm round shaft, 2m long cable)
* Quality, high-preload X-rail
